# AI Call Center Assistant

An interactive call simulation training tool built with HTML, CSS, and JavaScript.

## Project Structure

```
AI-CALL-CENTER-ASISTANT/
├── index.html          # Entry point (redirects to intro)
├── html/               # All HTML pages
│   ├── intro.html      # Welcome screen
│   ├── loading.html    # Loading screen
│   ├── main-menu.html  # Main menu
│   ├── call.html       # Active call simulation
│   └── result.html     # Call results page
├── css/                # Stylesheets
│   ├── intro.css
│   ├── loading.css
│   ├── main-menu.css
│   ├── call.css
│   ├── result.css
│   └── style.css
├── js/                 # JavaScript files
│   └── script.js
└── assets/             # Images and other assets
    └── bg.png
```

## How to Run

1. Open `index.html` in your web browser, or
2. Navigate directly to `html/intro.html`

## Features

- Interactive call simulation interface
- Real-time call timer
- Client information display
- Call transcript logging
- Performance evaluation system
- Responsive design

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
